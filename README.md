# Execute developer

1. Clone the repo
2. Execute
```
yarn install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```

4. Up DB
```
docker-compose up -d
```

5. Fill DB
```
http://localhost:3000/api/v2/seed
```
